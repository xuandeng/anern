# Anern Solar Energy Website Clone

一个完全复刻 [https://www.anern.com/](https://www.anern.com/) 的太阳能公司网站，使用现代化的前端技术栈构建。

## 🌟 项目特色

- **一比一复刻原网站**：完全遵循原网站的设计风格和用户体验
- **现代化技术栈**：使用 HTML5、Tailwind CSS 和 Alpine.js
- **响应式设计**：完美适配各种设备屏幕
- **交互丰富**：包含动态导航、表单验证、购物车等功能
- **性能优化**：使用 CDN 加载资源，快速加载

## 🛠️ 技术栈

- **HTML5**：语义化标记
- **Tailwind CSS**：实用优先的 CSS 框架
- **Alpine.js**：轻量级 JavaScript 框架
- **Font Awesome**：图标库

## 🚀 快速开始

### 本地运行

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/anern-website-clone.git
   cd anern-website-clone
   ```

2. **启动本地服务器**
   
   使用 Python：
   ```bash
   python -m http.server 8000
   ```
   
   或使用 Node.js：
   ```bash
   npx http-server
   ```

3. **访问网站**
   在浏览器中打开 `http://localhost:8000`

### 部署到生产环境

这是一个纯静态网站，可以部署到任何静态网站托管服务：

- **Netlify**：直接拖拽文件夹到 Netlify
- **Vercel**：连接 GitHub 仓库自动部署
- **GitHub Pages**：推送到 `gh-pages` 分支
- **传统服务器**：上传所有文件到 web 根目录

## 🚧 自定义和扩展

### 修改颜色主题
在每个 HTML 文件的 `<script>` 标签中修改 Tailwind 配置：

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                'anern-blue': '#your-color',
                'anern-light-blue': '#your-color',
                'anern-dark': '#your-color',
            }
        }
    }
}
```
